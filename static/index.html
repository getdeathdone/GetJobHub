How it works

Search first, index as you go

1

Choose sources

Select the job boards you want to check. The app only asks those sources, so a Djinni-only search stays Djinni-only.

2

Fetch fresh pages

Cloudflare Worker calls public APIs where available and lightweight HTML/RSS adapters for sites like Work.ua, DOU and Djinni.

3

Clean and match

Each vacancy is converted into one shape: title, company, location, salary, source link and a short description. Then it is scored against your query.

4

Save the index

Matched vacancies are stored in D1 and deduplicated by source URL. The overview shows this stored index, not every job that exists on the internet.
